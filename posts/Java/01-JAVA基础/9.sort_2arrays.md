# 排序、二维数组

## 堆和栈的简单介绍

 栈：一种先进后出的线性表

 堆：计算机科学中，一类特殊数据结构的统称

 * 存储：

 *   栈：存储的是局部变量   

 *   堆：new出来的东西   数组

 * 初始化：

 *   栈：栈中变量，必须先初始化，赋值，再使用

 *   堆：有初始值 默认值

 * 生存周期：

 *   栈：一旦用完，立刻销毁

 *   堆：一旦引用不存在，jvm回收

 * 响应速度：

 *   栈：响应速度 快

 *   堆：相对较慢

## 数组排序

  数组排序：

 *   冒泡排序：

 *   规则：

 *     大数据下沉，小数据上浮，如水中气泡一样，

 *     数组中元素两两比较，如果前面的大于后面的

 *     交换数据

 *  {5,7,1,3,2}   

 *  第一遍：

 *     5,7,1,3,2

 *     5,1,7,3,2

 *     5,1,3,7,2

 *     5,1,3,2,7

 *    

 *   通过第一次比较：

 *       最大值放到了最后

 *  第二遍：

 *     1,5,3,2,7

 *     1,3,5,2,7

 *     1,3,2,5,7

 *  第三遍:

 *     1,3,2,5,7

 *     1,2,3,5,7

 *  第四遍：

 *     1,2,3,5,7

 *  冒泡排序效率很低：

 *     如果有n个元素，比较n-1次
