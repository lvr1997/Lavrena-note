#  String、StringBuffer、API的使用

## 字符串String

 数据类型：
   基本数据类型
      4类8种：
      整数类型：`byte` `short` `int` `long`
      浮点数类型：`float` `double`
      字符类型：`char`
      布尔类型：`boolean`
   引用数据类型
      `String`
      在`java`中处理字符操作的方式：
      1、`Character` 字符类型 主要处理单个字符
      2、`String`  一串字符 不可变的
      3、`StringBuffer` 一串字符  可变的
   `String`：
      是对象，不是原始类型 
      是不可改变的对象，一旦被创建，就不能修改他的值
      一旦修改，就相当于重新创建
  `==`和`equals`(对象)：
   `==`：内存地址是否相等  也可以判断基本数据类型的值信息是否相等
    `equals`:比较的是内容是否相等
 字符串常用函数：
   1、获取方法
      `对象名.length()`  获取字符串的长度信息
      `对象名.charAt(int index)`:获取index位置上的字符信息
      `对象名.indexOf(int ch)`：获取ch第一次出现的位置
      `对象名.indexOf(int ch, int fromIndex);`
                  获取从哪个索引开始出现ch的位置
   2、判断方法

2、判断方法  `boolean ` `true` `false`

       `对象名.equals(字符串) `：判断两个字符串内容是否相等
       `对象名.isEmpty()`：判断字符串是否为空
       `对象名.startsWith(String str)`:判断字符串是否以str为开始
       `对象名.endsWith(String str)`判断字符串是否以str为结束
       `对象名.contains(str)`：是否包含某个字符串
   3、替换方法
   4、切割方法
   5、转换方法
   6、构造函数
   7、静态方法

 替换：
   `对象名.replace(char old, char new);`
         用`new` 代替字符串old所有的字符
   切割：
      `对象名.substring(int index)`：从哪个索引开始截取
      `对象名.substring(开始位置, 结束位置)`：从某个索引开始
            到某个索引结束，不包含结束位置
   转换：
         `对象名.split(String str)`:以str为分割，转成字符串数组    
 
    ...
    注意点：字符串索引从`0`开始到`n-1`结束，如果超出会报错

## StringBuffer应用

 `StringBuffer`：
   处理字符串信息，为可变的字符串
   如果添加扩展相当于在原来的信息上进行追加
   `String str ="abc"`
   `StringBuffer bf=new StringBuffer();`
   常用方法：
     `对象名.append(String str);`追加内容str到对象中
     `对象名.toString()`:把StringBuffer对象转成String
