# 面向对象

## 面向对象介绍

  面向过程：
     以过程为中心的编程思想  c
  面向对象：
     以对象为中心  java
  对象：现实生活中真实存在的事物 万物都是对象
  类：对同一对象行为和特征的抽取
  属性：对象的特征  变量
  方法：行为 执行的操作
 
  现实生活中  ：对象  => 类
  在java中 ：类 => 对象
  面向对象特征：
1. 封装
2. 继承
3. 多态

   注意点：
      在类中定义的变量信息可以不赋值直接使用
  成员变量：
      是属于对象的，是对象私有的，如果某个对象发生改变
      不会影响其他对象
  类变量：静态变量
      是属于类对象，属于所有对象所共有的，如果其中一个发生
      变化，其他的也会变化

4、this的用法

 this:关键字  指的是当前对象

  this的用法：
    1、通过this获取和方法内的同名的类中属性信息
       eg： this.属性名
    2、通过this可以在调用本类中其他的函数,这时候this
       可以省略不写
       eg：this.方法名(参数列表);
    3、通过this在构造函数调用其他构造函数
       eg:this(参数列表);
      
   注意点：
      1、构造函数不能递归调用
      2、this在构造函数中调用的时候只能放置在第一行
         所以不能多次调用，如果想多次调用，可以嵌套
         调用
      3、this不能在静态方法中使用
## 继承

    描述类之间的关系
  java类之间存在的关系：
   1、is a 关系 学生  人 一般和特殊  继承关系
   2、has a  包含的关系 整体和部分
*is a：一般 特殊 人 学生
   问题：
    1、没办法描述类之间关系
    2、会出现大量的重复性代码
  继承：
   结构体：
   修饰符 class 子类 extends 父类{
     
   }
   子类：继承其他类的类叫做子类 扩展类 继承类
   父类：被其他类继承的类叫做父类  基类
   继承：
     1、描述类之间的关系
     2、降低类之间的重复代码 提高开发效率
   注意点：
     1、子类并不能继承父类所有的属性和方法，比如私有的
     2、构造函数不能被继承
     3、java 中类只能被单继承
     4、所有的类都继承Object类，一旦有了自己继承的类
        默认继承的类就不存在了

重写：重写也叫做覆盖

       在继承关系中，子类可以定义和父类相同的名称
       且参数列表一致的函数，这种函数的体现就叫做
       函数的重写
  前提条件：
       必须有继承关系
  注意点：
       1、当子类重写了父类的函数，如果子类的对象
          调用该函数，一定调用的是重写后的函数
       2、子类重写父类的方法，权限大于等于父类的函数
          信息
  重载和重写的区别：
     重载：
       前提：必须在同一个类中
       特点：函数名相同  参数列表不同  与其他无关（返回类型 访问权限）
       不同： 个数不同 顺序不同  类型不同
    重写：
        前提：继承关系的存在 子类中
        特点：
            函数名相同 参数列表必须相同
            子类返回类型要小于等于父类的返回值类型
super:关键字 父类对象

  用法3种：
    1、获取父类中同名的属性信息  super.name
    2、获取被子类重写的父类中方法信息 super.方法名(参数列表)
    3、通过子类的构造函数获取父类的构造函数
       super(参数列表);
  注意点：
     1、子类会默认调用父类中无参数的构造函数
         一旦有了自己的调用，默认的就不存在了
     2、通过子类构造函数调用父类构造函数的时候必须
        放在第一行，调用只能调用一次
        this和super不能同时使用
     3、super不能在静态方法中使用
  this和super区别：
     1、使用
       this：在当前类中使用
       super：在继承关系的子类中使用
     2、特殊：
        this：指的是当前对象
        super：在子类中应用
