>       - 字符类型：`char`

>   - 引用数据类型：（1种）
>       - `String`

## 字符串String

在`java`中处理字符操作的方式：
1. `Character` 字符类型 主要处理单个字符
2. `String`  一串字符 不可变的
3. `StringBuffer` 一串字符  可变的

`String`：是对象，不是原始类型，是不可改变的对象，一旦被创建，就不能修改他的值， 一旦修改，就相当于重新创建

> [!tip] `==`和`equals`的区别
> `==`：内存地址是否相等  也可以判断基本数据类型的值信息是否相等
> `equals`:比较的是内容是否相等

### 字符串常用函数

::: code-group

```java [获取]
String str = "hello";

str.length()  //获取字符串的长度信息
str.charAt(int index) //获取index位置上的字符信息
str.indexOf(int ch) //获取ch第一次出现的位置
str.indexOf(int ch, int fromIndex); //获取从fromIndex开始出现ch的位置
```

```java [判断]
String str = "hello";

str.contains(str) //是否包含某个字符串
str.equals("字符串") //判断两个字符串内容是否相等
str.isEmpty() //判断字符串是否为空
str.startsWith(String str) //判断字符串是否以str为开始
str.endsWith(String str) //判断字符串是否以str为结束
```

```java [转换]
String str = "hello";
str.toLowerCase() //转成小写
str.toUpperCase() //转成大写
str.trim() //去掉字符串前后空格
```

```java [替换]
String str = "hello";
str.replace("ll", "MM") //将字符串中的ll替换成MM
```

```java [切割]
String str = "hello";

str.substring(2) //从索引6开始截取到结束
str.substring(0, 3) //从索引0开始截取到索引3结束，不包含索引3

// 注意点：字符串索引从`0`开始到`n-1`结束，如果超出会报错

```

```java [拼接]
String str = "hello";
str.concat("MM") //拼接字符串MM
```

```java [转换]
String str = "hello,world";

str.split(",") //以,为分割，转成字符串数组
```

:::

## StringBuffer应用

`StringBuffer`： 处理字符串信息，为可变的字符串，如果添加扩展相当于在原来的信息上进行追加

```java
String str ="abc"
StringBuffer bf=new StringBuffer();

// 常用方法：
bf.append(String str); // 追加内容str到对象中
bf.toString() //把StringBuffer对象转成String
```

> [!tip] String和StringBuffer区别
> - `String`：不可变的字符串对象
> - `StringBuffer`：可变的字符串对象
