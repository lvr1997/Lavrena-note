# 数组

同一种类型的集合，本身是一个容器
## 一维数组

可以给信息进行编号，方便操作数据

### 定义数组：

1、动态初始化
   数据类型 [] 变量名=new 数据类型[元素个数];
   分析：
    数据类型：基本数据类型或者引用数据类型
    []:代表数组
    变量名：只要符合命名规范即可
    左边也可以写成 ：数据类型 变量名[]
    右边：
      new:在内存给容器产生实体
      数据类型：必须和左边数据类型对应
      元素个数：能够存放几个元素
2、静态初始化
 静态初始化：
   数据类型 [] 变量名=new 数据类型[]{值1,值2,...};
   简写成以下格式:
   数据类型 [] 变量名={值1,值2,值3 };
 遍历方式：
   for(数据类型  变量名:要遍历数组的变量名){
      syso(变量名);
   }
  
 主要用在数组查询的时候
  
   注意点：
     数组下标从0开始到n-1结束
   默认值：
     int 0
     double 0.0
     String null
     ...
